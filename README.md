# Mój projekt strony internetowej

## Opis

Ze względu na prywatność, w projekcie zostały użyte przypadkowe zdjęcia znalezione w Internecie.  
Również niektóre nazwy i dane zostały celowo usunięte lub zastąpione kropkami (`...`), aby nie ujawniać prywatnych informacji.

---

> Projekt stworzony w celach demonstracyjnych.

# My Website Project

## Description

For privacy reasons, this project uses random images found on the Internet.  
Some names and data have also been intentionally removed or replaced with dots (`...`) to avoid revealing private information.
